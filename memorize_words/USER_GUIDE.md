# Word Memorizer — User Guide

## Getting Started

1. When you open the app, pick your name or create a new user.
2. Each user has their own separate word list and progress.
3. Use **Switch User** in the nav bar to change users.

## Adding Words

- Go to **Add Word** and type a word (definition is optional).
- To bulk-add, import a text file with one `word|definition` per line.

## Reviewing

- Go to **Review** to study cards that are due today.
- Tap a card to reveal the definition, then rate yourself:
  - **Know** → longer interval before next review
  - **Don't Know** → card comes back sooner
- Use **Review All** to practice everything, not just due cards.

### Keyboard Shortcuts (Review screen)

| Key | Action |
|-----|--------|
| Space / ↑ / ↓ | Flip card |
| → | Know |
| ← | Don't Know |

## Tracking Progress

- **Stats** shows your total words, due today count, and mastery %.
- **My Words** lists all words sorted by review date, with status badges: *Mature*, *Due*, or *Upcoming*.

## Housekeeping

- Delete individual words from **My Words**.
- **Reset All Progress** keeps your words but restarts all review schedules.
- **Delete All Words** wipes everything for the current user.
